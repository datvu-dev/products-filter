# products-filter

This is a simple page that shows and filters products.

### Install packages
```bash
npm install
```

### Install http-server
```bash
npm install -g http-server
```

### Generate main.js
```bash
browserify ./src/js/scripts.js > ./src/main.js
```

### Run app
```bash
http-server
```

### Run test
```bash
npm run test
```
